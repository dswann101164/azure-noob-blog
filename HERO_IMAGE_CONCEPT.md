# Hero Image Concept: Software Rationalization

## Image Name
`rationalization-devops.png`

## Dimensions
1200 x 630 px (optimal for social sharing)

## Visual Concept

### Center Focus
- Large, glowing **AppID tag** (golden/amber color)
- Text on tag: `AppID: APP-001`
- Slight 3D depth with shadow

### Concentric Rings Around AppID
Three security zone rings (like planets orbiting):
1. **Inner ring (Dev)**: Light blue, dashed line
2. **Middle ring (Test)**: Orange, dashed line  
3. **Outer ring (Prod)**: Red, solid line

Each ring labeled with subscription icon + text:
- "sub-dev-finance"
- "sub-test-finance"
- "sub-prod-pci"

### Background Elements
- Faded Azure subscription IDs scattered (very subtle)
- Matrix-style code rain effect (light)
- Azure blue gradient (#0078D4 → darker)

### Bottom Right Corner
- Small pile of **burning/crossed-out** VMs
- Labels: "Untagged", "Orphaned", "Zombie"
- Fire emoji or flames

### Bottom Banner (Across)
Text: **"Truth > Structure"**
- Bold, clean font
- White text on semi-transparent dark bar

### Color Scheme
- Primary: Azure blue (#0078D4)
- Accent: Golden amber for AppID tag
- Security zones: Blue → Orange → Red gradient
- Background: Dark blue to black gradient
