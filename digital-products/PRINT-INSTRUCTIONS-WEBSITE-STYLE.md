## Step 3: Print to PDF

### Chrome (Recommended):
1. Press **Ctrl+P** (or File → Print)
2. **Destination:** "Save as PDF"
3. **Paper size:** Letter
4. **Layout:** Portrait
5. **Margins:** Default
6. **Scale:** 100%
7. **Options:**
   - ✅ Background graphics (optional - document already has clean print styling)
8. Click **Save**

### Edge:
1. Press **Ctrl+P**
2. **Printer:** Microsoft Print to PDF
3. **Layout:** Portrait
4. **Paper size:** Letter
5. **Scale:** 100%
6. Click **Print**
